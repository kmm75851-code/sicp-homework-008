### `make_counter(start=0, step=1)`

Возвращает функцию-счётчик. При каждом вызове увеличивает значение на `step`.

- `start` — начальное значение (до первого вызова)
- `step` — шаг увеличения

```python
c = make_counter()
c()  # → 1
c()  # → 2
c()  # → 3

c2 = make_counter(start=10, step=5)
c2()  # → 15
c2()  # → 20

# Два счётчика независимы
a = make_counter()
b = make_counter()
a()  # → 1
a()  # → 2
b()  # → 1  ← свой счётчик!
```

💡 *Подсказка:* `nonlocal count`, `count += step`.

---

### `make_accumulator(initial=0)`

Возвращает функцию, которая при каждом вызове **прибавляет** аргумент к накопленной сумме и возвращает её.

```python
acc = make_accumulator(5)
acc(10)   # → 15
acc(10)   # → 25
acc(-5)   # → 20
acc(0)    # → 20

acc2 = make_accumulator()
acc2(3)   # → 3
acc2(3)   # → 6
```

---

### `make_history_counter()`

Возвращает **две** функции: `(tick, history)`.

- `tick()` — увеличивает счётчик, ничего не возвращает
- `history()` — возвращает **список** всех значений после каждого `tick`, от первого к последнему

```python
tick, history = make_history_counter()
tick()
tick()
tick()
history()   # → [1, 2, 3]
tick()
history()   # → [1, 2, 3, 4]

tick2, history2 = make_history_counter()
tick2()
history2()   # → [1]   ← независимо от первого
```

💡 *Подсказка:* внутри храни `count` и `log = []`. `tick` делает `nonlocal count`, увеличивает, добавляет в log. `history` возвращает копию `log[:]`.

---

### `make_adder()`

Возвращает функцию `add(n)`. Каждый вызов добавляет `n` к сумме и возвращает новую сумму. Отдельно возвращает функцию `reset()`, обнуляющую сумму.

Возвращает кортеж `(add, reset)`.

```python
add, reset = make_adder()
add(5)    # → 5
add(3)    # → 8
add(2)    # → 10
reset()
add(1)    # → 1
```

---

### `make_minmax_tracker()`

Возвращает функцию `track(n)`. При каждом вызове фиксирует число.
Возвращает `(минимум, максимум)` из всех зафиксированных чисел.
Если вызвана без аргумента (`track()`), возвращает текущий `(min, max)` без добавления числа.

```python
track = make_minmax_tracker()
track(5)    # → (5, 5)
track(2)    # → (2, 5)
track(9)    # → (2, 9)
track(3)    # → (2, 9)
track()     # → (2, 9)  ← без аргумента, просто смотрим
```

💡 *Подсказка:* используй дефолтный аргумент — `def track(n=None)`. Если `n is None` — просто верни текущие. Иначе — обнови `nonlocal min_val, max_val`.

## Часть 2: Словари

### Файл: `dicts.py`

---

### `word_count(text)`

Принимает строку. Возвращает словарь `{слово: количество_вхождений}`.
Слова разделяются пробелами. Регистр не важен: `"Hello"` и `"hello"` — одно слово.

```python
word_count("the cat sat on the mat")
# → {'the': 2, 'cat': 1, 'sat': 1, 'on': 1, 'mat': 1}

word_count("Hello hello HELLO")
# → {'hello': 3}

word_count("")
# → {}
```

💡 *Подсказка:* `text.lower().split()`. Используй `d.get(word, 0) + 1`.

---

### `group_by_length(words)`

Принимает список слов. Возвращает словарь `{длина: [слова такой длины]}`.
Порядок слов внутри каждой группы — как в исходном списке.

```python
group_by_length(["hi", "hello", "ok", "world", "yes"])
# → {2: ['hi', 'ok'], 5: ['hello', 'world'], 3: ['yes']}

group_by_length([])
# → {}
```

---

### `invert_dict(d)`

Инвертирует словарь: ключи становятся значениями, значения — ключами.
Если несколько ключей имеют одно значение — в результате значение маппится на **список** всех исходных ключей.

```python
invert_dict({'a': 1, 'b': 2, 'c': 1})
# → {1: ['a', 'c'], 2: ['b']}

invert_dict({'x': 10, 'y': 20})
# → {10: ['x'], 20: ['y']}

invert_dict({})
# → {}
```

---

### `merge_dicts(*dicts)`

Принимает любое количество словарей. Возвращает новый словарь, объединяющий все.
Если ключ встречается в нескольких словарях — берётся значение из **последнего** словаря.

```python
merge_dicts({'a': 1}, {'b': 2}, {'a': 99})
# → {'a': 99, 'b': 2}

merge_dicts({'x': 1})
# → {'x': 1}

merge_dicts()
# → {}
```

---

### `most_common(text, n)`

Принимает текст и число `n`. Возвращает список из `n` самых частых слов в порядке убывания частоты.
Если слов с одинаковой частотой несколько, порядок между ними произвольный.

```python
most_common("the cat sat on the mat the cat", 2)
# → ['the', 'cat']

most_common("a b c", 2)
# → два из ['a', 'b', 'c'] в любом порядке

most_common("hello", 5)
# → ['hello']  ← слов меньше чем n — вернуть все
```

💡 *Подсказка:* посчитай частоты через `word_count`, потом отсортируй `sorted(freq.items(), key=lambda x: -x[1])`.

---

### `two_sum(nums, target)`

Принимает список чисел и цель. Возвращает кортеж `(i, j)` — **индексы** двух чисел, которые в сумме дают `target`.
Гарантируется ровно одна пара. `i < j`.

```python
two_sum([2, 7, 11, 15], 9)   # → (0, 1)   # 2 + 7 = 9
two_sum([3, 2, 4], 6)        # → (1, 2)   # 2 + 4 = 6
two_sum([1, 5, 3, 7], 8)     # → (1, 3)   # 5 + 7 = 8  ← не (0,2): 1+3=4
```

💡 *Подсказка:* словарь `seen = {значение: индекс}`. Для каждого числа смотришь, есть ли `target - num` в `seen`.

---

### `char_frequency(s)`

Возвращает словарь `{символ: количество}` для всех символов строки (включая пробелы)

```python
char_frequency("hello")
# → {'h': 1, 'e': 1, 'l': 2, 'o': 1}

char_frequency("aaa")
# → {'a': 3}

char_frequency("")
# → {}
```

---

### `index_builder(documents)`

Строит **обратный индекс**: словарь `{слово: set(id документов)}`.
Принимает словарь `{id: текст}`.

```python

docs = {
    1: "the cat sat",
    2: "the cat on the mat",
    3: "the dog sat"
}
index_builder(docs)
# → {
#     'the': {1, 2, 3},
#     'cat': {1, 2},
#     'sat': {1, 3},
#     'on':  {2},
#     'mat': {2},
#     'dog': {3}
# }
```

💡 *Подсказка:* для каждого `(doc_id, text)` разбей текст, для каждого слова добавь `doc_id` в `index[word]` (используй `.setdefault(word, set())`).

## Часть 3: Множества

### Файл: `sets.py`

---

### `unique_words(text)`

Возвращает множество уникальных слов (нижний регистр, разделитель — пробел).

```python
unique_words("the cat sat on the mat")
# → {'the', 'cat', 'sat', 'on', 'mat'}

unique_words("Hello hello HELLO")
# → {'hello'}
```

---

### `common_words(text1, text2)`

Возвращает множество слов, встречающихся **в обоих** текстах.

```python
common_words("the cat sat", "the dog sat")
# → {'the', 'sat'}


common_words("hello world", "foo bar")
# → set()
```

---

### `only_in_first(text1, text2)`

Возвращает множество слов, которые есть в `text1`, но **нет** в `text2`.

```python
only_in_first("the cat sat on the mat", "the dog sat")
# → {'cat', 'on', 'mat'}
```

---

### `dedup_preserve_order(items)`

Удаляет дубликаты из списка, сохраняя **порядок первого вхождения**.
Запрещено использовать `dict.fromkeys`.

```python
dedup_preserve_order([3, 1, 2, 1, 3, 4])   # → [3, 1, 2, 4]
dedup_preserve_order(['a', 'b', 'a', 'c'])  # → ['a', 'b', 'c']
dedup_preserve_order([])                     # → []
```

💡 *Подсказка:* `seen = set()`, добавляй в результат только если элемент ещё не в `seen`.

---

### `anagram_groups(words)`

Группирует слова-анаграммы вместе. Возвращает список **отсортированных** групп (каждая группа — отсортированный список).
Группы, в которых один элемент, тоже включаются.

```python
anagram_groups(["eat", "tea", "tan", "ate", "nat", "bat"])
# → [['ate', 'eat', 'tea'], ['bat'], ['nat', 'tan']]
# (порядок групп не важен)


anagram_groups(["abc", "bca", "xyz"])
# → [['abc', 'bca'], ['xyz']]
```

💡 *Подсказка:* ключ группы — `tuple(sorted(word))`. Используй словарь `{ключ: [слова]}`.

---

### `set_stats(items)`

Принимает список чисел. Возвращает словарь со статистикой:

- `unique_count` — количество уникальных значений
- `duplicates` — множество значений, которые встречаются **более одного раза**
- `unique_only` — множество значений, которые встречаются **ровно один раз**

```python
set_stats([1, 2, 2, 3, 3, 3, 4])
# → {
#     'unique_count': 4,
#     'duplicates': {2, 3},
#     'unique_only': {1, 4}
# }

set_stats([1, 2, 3])
# → {'unique_count': 3, 'duplicates': set(), 'unique_only': {1, 2, 3}}
```

## Часть 4: Финальный проект — Text Analyzer

### Файл: `text_analyzer.py`

Объединяет все три темы. Одна большая функция `analyze(text)`.

---

### `analyze(text)`

Принимает текст (строку). Возвращает словарь со всеми метриками:

```python
result = analyze("the cat sat on the mat the cat")

result == {
    seen = {}
    
    
    for x in items:
    'total_words': 8,
    'unique_words': 5,                     # {'the', 'cat', 'sat', 'on', 'mat'}
    'word_counts': {                       # все слова и частота
        'the': 3, 'cat': 2,
        'sat': 1, 'on': 1, 'mat': 1
    },
    'most_common': 'the',                  # самое частое слово
    'longest_word': 'mat',                 # самое длинное (первое при равенстве)
    'avg_word_length': 3.0,                # средняя длина слова (уникальных)
    'appears_once': {'sat', 'on', 'mat'},  # слова с частотой 1
}
```

Требования:
- Слова в нижнем регистре
- `most_common` — одно слово (строка), не список
- `longest_word` — если несколько слов одной длины, берётся первое по алфавиту
- `avg_word_length` — среднее по **уникальным** словам, округление через `round(x, 2)`

---

### `make_analyzer()`

Возвращает функцию-анализатор **с памятью**: накапливает все переданные тексты.

- `add(text)` — добавляет текст в накопленный корпус
- `stats()` — возвращает `analyze(...)` по всему накопленному тексту

```python
add, stats = make_analyzer()
add("the cat sat")
add("the dog ran")
stats()
# analyze("the cat sat the dog ran") — по всему накопленному
```

💡 *Подсказка:* `nonlocal corpus`, `corpus += " " + text`.
